ICEWorldPage

  • This is the main windows for our program you will see my button such as chat, yell, zoom in, zoom out, customization, File Menu bar, LogOut
  • Chat : insert your text in the following text box at the bottom of the screen and it will print it above your character in the screen
  • Yell : Alike chat button this will print out a large oval colored text in the middle of your screen and it will last for about 10 seconds
  • Customization : When you press this button it will automatically redirect you to another screen this screen will customized your character with weapon, helmet, armor and shield
  • Zoom in and Zoom Out : As the button tell it use to zoom the screen in and out
  • Log Out : use to close the program
  • Sound Setting : this is a controller of the audio in the program
